# Smart Learning Planner

A modern productivity and study management web application built using React and Tailwind CSS. The application helps students manage tasks, track progress, and improve learning productivity through an interactive dashboard.

---

# Features

- Add new study tasks
- Delete tasks
- Mark tasks as completed
- Dynamic task counters
- Responsive dashboard UI
- Task progress tracking
- Clean and modern design
- Built with React Hooks

---

# Installation

## Clone Repository

```bash
git clone https://github.com/your-username/smart-learning-planner.git
```

## Open Project

```bash
cd smart-learning-planner/frontend
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```
